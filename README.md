# 🏋️ NEXUS Store — Full Stack E-Commerce

A complete clothing brand e-commerce platform with storefront, admin dashboard, and Egyptian payment methods.

## 📞 Features

- ✅ Responsive storefront (mobile-first)
- ✅ Product listing with filters (category, gender, sort, search)
- ✅ Product detail with size/color selector
- ✅ Shopping cart with persistent storage
- ✅ 3-step checkout flow
- ✅ Visa/MasterCard via Stripe
- ✅ Vodafone Cash via Paymob
- ✅ InstaPay via Paymob
- ✅ Admin dashboard with stats & charts
- ✅ Product CRUD (add/edit/delete)
- ✅ Order management with status updates
- ✅ Customer list
- ✅ JWT authentication
- ✅ Image upload
## 🎯 Goals

Build a modern and scalable fashion eCommerce experience with clean design and optimized performance.
